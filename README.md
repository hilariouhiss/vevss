# 如何使用

## 版本
1. 项目框架: `Spring Boot:3.2.1`.
2. 数据库: `MySQL:8`, 使用 `Spring Data JPA` 操作数据库.
3. 依赖管理: `Maven:3.9.6`.
4. 项目启动端口: `8080`.
5. JDK 版本: `Oracle-JDK21`.
6. 使用Docker部署.

## 使用方法
1. 部署: 运行 `docker-compose up -d` 即可, 等待一段时间即可完成部署.
2. 停止: 运行 `docker-compose down` 即可.
3. 访问: 访问 `http://localhost:8080` 即可.
4. 发射点法发
